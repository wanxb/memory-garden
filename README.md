# Memory Garden

A poetic interactive web page where a short sentence can be planted into a night sky and grow into a glowing micro-plant.

## Overview

Memory Garden is a standalone static web artwork built around a very simple interaction: click, type a sentence, and watch it become part of a luminous digital garden. It is not a productivity tool or a game with a score. It is a gentle emotional object for pauses, mood, and atmosphere.

## Highlights

- Plant custom text directly into the scene
- Generate random poetic lines instantly
- Trigger a meteor shower for extra ambience
- Copy the current sentence to clipboard
- Export the current canvas as an image
- Fully static, no backend required

## Positioning

This project works well as:
- an interactive art page
- a poetic landing experiment
- a mood-based micro experience
- a portfolio piece with emotional tone
